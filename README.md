A beginner-friendly Java project implementing a reservation system using OOP concepts, Array List, and Scanner for console interaction.
# Reservation System (Java)

This is a simple console-based reservation system developed in Java.

## Features
- Make a reservation
- View all reservations
- Cancel a reservation
- Automatically generated reservation IDs

## Technologies Used
- Java
- OOP (Classes and Objects)
- ArrayList
- Scanner

## How to Run
1. Compile the program:
   javac ReservationSystemUI.java
2. Run the program:
   java ReservationSystemUI
